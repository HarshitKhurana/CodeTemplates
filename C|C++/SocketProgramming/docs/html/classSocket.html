Detailed Description

Class responsible for managin low-level sockets.

This class encapsulates some low-level socket functionalities and provide an easy to use interface for handling the sockets. Some of the functions are static becuase they doesn't require access to any class member and moreover need to be able to access these seperately. This function will put the response in the host or else will nullptr.
